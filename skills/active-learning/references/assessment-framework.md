# Self-Assessment Framework

AI 自我评估学习成果的简单框架。

## Quick Learning Assessment

任务完成 = 学习完成。无需额外评估。

## Practical Mastery Checklist

```
□ Can I complete typical tasks without looking up docs?
□ Do I understand the core concepts?
□ Have I noted the important gotchas?
```

If any unchecked → identify gap → search specifically → update memory.

## Deep Mastery Checklist

```
□ Can I explain the core concepts clearly?
□ Can I work independently without docs?
□ Can I handle edge cases?
□ Can I debug common issues?
□ Can I make good design decisions?
```

If gaps → search → practice through real work → update memory.

## Assessment During Work

最有效的评估是在实际工作中：

1. **完成任务** — 能完成说明基本掌握了
2. **遇到问题** — 记录下来，事后补充到记忆
3. **需要查文档** — 查完把关键信息沉淀下来

## When to Stop

- **Quick**: 任务完成
- **Practical**: 能独立完成常见任务
- **Deep**: 能处理边界情况和调试问题

**不要追求完美**。目标是"够用，知道去哪查"。
